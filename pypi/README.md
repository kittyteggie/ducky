# Ducky 🦆

> **The future of Python starts here.**

`printducky` is a groundbreaking Python library focused on solving one of the most overlooked problems in software engineering: printing a duck to the console.

Although the project is currently small, its long-term vision is nothing short of revolutionary. We firmly believe that future Python documentaries, programming courses, and software engineering textbooks will have no choice but to dedicate entire chapters to the impact of `printducky`.

## Installation

```bash
pip install printducky
```

## Usage

```python
import printducky
```

That's it.

A duck will appear.
